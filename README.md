## Run Instructions

1. Clone the repository on to local device
2. Open the Sleep Journal Project in Xcode
3. Choose an *iOS 26.1 Simulator* as the run target
4. Press 'Cmd + R' to build and run the app on the simulator
5. Allow location and photo library access when prompted if you want to test all entry creation features. 

No extra setup, package installation, or backend configuration is needed
